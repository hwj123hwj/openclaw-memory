## 三、经验总结

### 技能开发

1. **SKILL.md 必须清晰**：子 Agent 需要明确的执行指令
2. **脚本必须有执行权限**：`chmod +x scripts/*.sh`
3. **数据源优先级**：Tavily > 必应中文 > 今日头条 > 其他

### API 使用

1. **Tavily API Key 配置**：`~/.claude/settings.json`
2. **微博/今日头条**：需要浏览器自动化，纯 curl 无法获取
3. **热榜 API**：今日头条、百度热搜可以直接 curl 获取 JSON

### 子 Agent 协作

1. 子 Agent 需要明确的任务描述
2. 可以指定执行脚本路径
3. 建议设置合理的超时时间（180-300秒）
