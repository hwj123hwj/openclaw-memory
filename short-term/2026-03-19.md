## 技术经验

### 邮箱 IMAP 连接 (Python)
```python
import imaplib, ssl
context = ssl.create_default_context()
mail = imaplib.IMAP4_SSL("imap.qq.com", 993, ssl_context=context)
mail.login("邮箱", "授权码")
mail.select("INBOX")
```

### 订阅退订方式
1. HTTP 退订链接：直接 curl 访问
2. 邮件退订：发送邮件到 List-Unsubscribe 地址
3. 无退订链接：需手动去官网关闭
